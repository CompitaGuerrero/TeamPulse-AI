# TeamPulse AI

TeamPulse AI — мини-дашборд для команды, который превращает рабочий вопрос в структурированный ответ: краткий вывод, разбор, риски и следующие действия. Пользователь может выбрать режим ответа: `Кратко`, `Подробно` или `План действий`.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- OpenAI Chat Completions API через server route
- localStorage для истории последних 5 запросов

## Features

- Рабочий вопрос через `textarea` с валидацией пустого ввода.
- Переключатель режимов ответа: `Кратко`, `Подробно`, `План действий`.
- Отправка кнопкой `Спросить AI` или через `Cmd/Ctrl + Enter`.
- Server route `POST /api/ask`, который скрывает API-ключ от клиента.
- Специализированный system prompt для командного помощника с адаптацией под выбранный режим.
- Server-side проверка непонятного ввода: случайные символы и сообщения без ясного запроса получают простой ответ без структурированной разбивки.
- Loading, empty, success и error states.
- История последних 5 запросов с сохранением вопроса, ответа и режима в `localStorage`.
- Восстановление ответа из истории по клику.
- Удаление отдельной записи из истории и кнопка `Очистить историю`.
- Копирование ответа через Clipboard API с обратной связью.
- Человеческие ошибки для отсутствующего ключа, API failures, сетевых проблем и timeout.

## Architecture Notes

- `src/app/api/ask/route.ts` отвечает за LLM-интеграцию, timeout, режим ответа и безопасные ошибки без stack trace.
- `src/lib/aiPrompt.ts` хранит system prompt отдельно от UI и собирает режимные инструкции.
- `src/lib/inputClarity.ts` отсеивает бессмысленные или слишком неясные сообщения до обращения к LLM.
- `src/lib/responseModes.ts` содержит общий список режимов для клиента и server route.
- `src/lib/storage.ts` изолирует работу с `localStorage` и не ломает SSR.
- `src/components/*` разбивают интерфейс на форму, ответ, историю, ошибку, empty state и copy button.
- Ответ AI не выводится как JSON: UI мягко форматирует обычный структурированный текст.
